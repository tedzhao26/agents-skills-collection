# Notion Skills

Comprehensive workflows for Notion workspace automation.

## Usage

- `/notion meeting prep for [topic]` - Gather context and create meeting materials
- `/notion research [topic]` - Search, synthesize, and document findings
- `/notion capture` - Turn this conversation into a wiki page
- `/notion spec-to-tasks [page]` - Transform specs into actionable tasks

## Capabilities

| Workflow | Description |
|----------|-------------|
| **Meeting Intelligence** | Prep materials with Notion context + research |
| **Research & Documentation** | Search, synthesize, create structured reports |
| **Knowledge Capture** | Conversations → wiki pages, decision records |
| **Spec to Implementation** | Specs → tasks with progress tracking |

## Notion Tools

| Tool | Purpose |
|------|---------|
| `notion-search` | Find pages and databases |
| `notion-fetch` | Read page/database content |
| `notion-create-pages` | Create new pages |
| `notion-update-page` | Update existing pages |

## Examples

```
/notion meeting prep for Q1 planning
/notion research competitor analysis for [product]
/notion capture this debugging session
/notion spec-to-tasks the auth redesign spec
```
